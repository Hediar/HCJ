### 태그

<> ~ </> 로 작성하며 안에 들어간 것을 특정한 명령 수행

- 기본 태그
    
    <!DOCTYPE html> : 문서 유형을 html로 지정
    
    <html lang="ko"> : 문서를 html로 시작, 언어를 한국어로 지정
    
    <head> : 주로 브라우저의 정보를 입력
    
    <body> : 문서 내용 입력
    
- 문서 구조 태그
    
    <header>
    
    <main>
    
    <section>
    
    <aside>
    
    <footer>
    
    <nav>
    
    <article>
    
    <section>
    
    <div>
    
- 내용 입력
